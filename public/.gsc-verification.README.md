# Google Search Console — HTML file verification

To verify domain ownership using the HTML file method:

1. In GSC, choose "URL prefix" property type (not "Domain") and choose
   "HTML file" as the verification method.
2. Download the file Google provides (e.g., `google1234abcd5678.html`).
3. Place it directly in this `public/` folder — same level as this README.
4. Commit, push, and deploy.
5. After deployment, verify the file is reachable at
   `https://yuvaldigital.co.il/<filename>.html`.
6. Click "Verify" in GSC.

Once verified, the file should remain in place — Google re-checks it
periodically. Do not delete it.

This README itself is harmless — it sits in `public/` but is never
rendered as a route, and Google ignores files starting with a dot.
